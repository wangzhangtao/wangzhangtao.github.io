安装docker

编辑安装脚本

vim /data/shell/install_docker.sh

远程执行安装脚本

查看docker系统资源

官网文档 docker system

df: 查看docker磁盘使用情况

提供Docker整体磁盘使用率的概况，包括镜像、容器和（本地）volume。所以我们现在随时都可以查看Docker使用了多少资源。

info: 查看系统信息

查看整个docker系统的信息, 这个命令的缩写为docker info

events: 从服务器获取实时事件

缩写命令 docker events

  • -f “image”=”mysql:5.6”
  • –since ：从指定的时间戳后显示所有事件;
  • –until ：流水时间显示到指定的时间为止；

prune: 清理磁盘

谨慎操作，镜像和容器将会被批量清理

  • -f 强制删除，不再询问
  • -a 所有没有被使用的镜像都删除
